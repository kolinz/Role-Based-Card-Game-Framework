# クイックスタートガイド

## 🚀 3ステップで始める

### ステップ1: インストール
```bash
npm install
```

### ステップ2: データベース初期化
```bash
npm run initdb
```

**注意**: このステップは**オプション**です。
`npm start`で起動すると自動的にデータベースが作成されます。

明示的に初期化したい場合や、データベースをリセットしたい場合のみ実行してください。

### ステップ3: サーバー起動
```bash
npm start
```

ブラウザで開く: http://localhost:3000

**管理画面**: http://localhost:3000/admin.html (admin / admin123)

---

## 📖 詳細な使い方

### データベースについて

#### 自動初期化（推奨）
```bash
npm start
```
初回起動時に自動的に:
- テーブルが作成される
- サンプルデータが投入される

#### 手動初期化
```bash
npm run initdb
```
以下の場合に使用:
- データベースをリセットしたい
- 既存のデータを削除したい
- クリーンな状態から始めたい

## 🎮 ゲームの流れ

### 1. ゲーム作成
1. ブラウザで http://localhost:3000 を開く
2. 「新しいゲームを作成」
3. プレイヤー名と最大人数を入力
4. 招待URLをコピー

### 2. 友達を招待
- 招待URLを友達に送る
- または、セッションIDを共有

### 3. ゲーム開始
- 全員が揃ったら「ゲーム開始」
- サイコロを振る
- カードを選ぶ
- 目標達成で勝利！

## 💡 トラブルシューティング

### ポート3000が使用中
```bash
PORT=8080 npm start
```

### データベースをリセット
```bash
npm run initdb
npm start
```

### 接続できない
1. ファイアウォール設定を確認
2. サーバーが起動しているか確認
3. ブラウザのコンソールでエラーを確認
