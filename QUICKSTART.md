# クイックスタートガイド

## 🚀 3ステップで始める

### ステップ1: インストール
```bash
npm install
```

### ステップ2: データベース初期化
```bash
npm run initdb
```

### ステップ3: サーバー起動
```bash
npm start
```

ブラウザで開く: http://localhost:3000

**管理画面**: http://localhost:3000/admin.html (admin / admin123)

---

## 🎮 ゲームの流れ

### 1. ゲーム作成
1. ブラウザで http://localhost:3000 を開く
2. 「新しいゲームを作成」
3. プレイヤー名と最大人数を入力
4. 招待URLをコピー

### 2. 友達を招待
- 招待URLを友達に送る
- または、セッションIDを共有

### 3. 職種選択
- 全員が好きな職種カードを選択

### 4. ゲーム開始
- サイコロを振る
- カードを選ぶ
- 目標達成で勝利！

## 🔧 コマンド一覧

| コマンド | 説明 |
|---------|------|
| `npm install` | 依存関係をインストール |
| `npm run initdb` | データベースを初期化 |
| `npm start` | サーバーを起動 |
| `npm run dev` | 開発モードで起動（自動リロード） |
